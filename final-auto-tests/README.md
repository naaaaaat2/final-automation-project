# Финальный проект по автоматизации тестирования

## Описание проекта
Автоматизация UI и API тестов для сервиса YouGile (или другого согласованного проекта).

## Структура проекта
- `config/` - конфигурационные файлы и тестовые данные
- `pages/` - Page Object модели для UI тестов
- `api/` - API клиент и endpoints
- `tests/` - тестовые файлы
- `utils/` - вспомогательные утилиты
